SQL

SQL (Structured Query Language) is a domain-specific language for managing data held in a DMS. QL is very flexible and allows users to access multiple tables at the same time. SQL became a standard of the American National Standards Institute (ANSI) in 1986.

SQL allows for CRUD operations and more. The CRUD operations are the four main operations required for data persistence. CRUD stands for; create, read, update and delete. SQL has various commands that enable these basic operations. SQL is used with relational databases and as such SQL has CRUD operations for managing tables.

Tables in SQL can be created using the CREATE TABLE command.

A live SQL database will have multiple tables of structured data that have relationships with each other. Relations are stored where "keys" in rows match between different tables. For example, a table 'person' might have 'favouriteFood' column and that stores the id of a food item in the 'food' table.

Retrieving data where the two id match can be achieved using a JOIN statement.

The most common SQL commands are:

SELECT - extracts data from a database UPDATE - updates data in a database DELETE - deletes data from a database INSERT INTO - inserts new data into a database CREATE DATABASE - creates a new database ALTER DATABASE - modifies a database CREATE TABLE - creates a new table ALTER TABLE - modifies a table DROP TABLE - deletes a table CREATE INDEX - creates an index (search key) DROP INDEX - deletes an index

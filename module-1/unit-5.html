Comparing and contrasting Operating systems and an application's features

Operating System: Windows 10

Application: Chrome

The Windows OS provides facilities to launch programs, store and access files. The Chrome web-browser facilitates loading and view web files. The Chrome browser is an optional application available on the Windows operations system, amongst others. The Windows OS on the other hand runs on hardware, not another software stack. The windows operating system provides first-class functionality to both users and developers, for example, file creates for users and an API for developers to leverage. The chrome browser is primarily for users, though does provide an API for developers to create extensions.

While the extensions for Chrome are primarily for extending the features related to browsing, new applications for Windows may extend functionality to provide almost infinite different types of features. As such the API's provided by Windows are broad and offer an interface that can facilitate many use cases. The Chrome API however is tailored specifically for extending its exists feature set.
